# M1 Signal Decoder — 第一性原理文档

> **M1 的任务只有一个：忠实地把世界上已经发生的变化，翻译成结构化语言。**

---

## 存在意义

### M1 解决的根本问题

人类每天面对的市场信息洪流中，99% 是噪音。

噪音不是"错误的信息"，而是对交易决策没有增量价值的信息。"恒大债务出问题了"是信息，但如果市场已经因此下跌了30%，这个信息对你今天的决策已经没有边际价值——它是已定价的。"某某分析师认为A股明年将涨20%"是预测，不是信号——没有人能保证它对。

**M1 存在，是为了把"已发生的客观变化"从噪音中分离出来，翻译成可以被后续模块处理的结构化单元（MarketSignal）。**

M1 不判断这个变化是否构成机会，不预测未来走向，不提供投资建议。它只做一件事：**如实记录已经变了什么。**

---

## 第一性原理

### 什么是"有效信号"？

有效信号 = **已发生的事实变化** × **尚未被市场充分定价** × **对某个市场/行业/标的有明确的方向性影响**

三个要素缺一不可：

**要素1：已发生的事实变化**
- 必须是"已经发生"的事，不是"可能会发生"的事
- 必须是"变化"，不是"静止的状态"——"中国GDP约为美国的70%"不是信号，"中国GDP增速从5.2%降至4.6%"是信号
- 必须有具体的证据支撑（数据、公告、官方文件）

**要素2：尚未充分定价**
- 如果信息已经是"地球人都知道"，市场大概率已经定价
- 判断依据：发生时间（越新越可能未定价）、信息传播范围（越小众越可能未定价）
- 这对应 `timeliness_score` 字段

**要素3：方向性影响明确**
- 信号必须对某个具体的市场/行业/标的有可以推断的方向性影响
- "某某公司员工离职率升高"可能是信号，但方向不够明确
- "某某公司宣布大规模裁员30%"是信号，方向明确（短期看空公司股价）

### 信号 vs 噪音：根本区别

| 维度 | 信号 | 噪音 |
|------|------|------|
| 事实基础 | 已发生的客观变化 | 预测、情绪、观点 |
| 方向性 | 对特定标的有明确影响方向 | 影响不明确或相互抵消 |
| 增量价值 | 对市场认知有增量 | 已知信息的重复 |
| 可证伪性 | 有具体数据和来源 | 无法被核实 |

## 设计约束

M1 必须严格遵守以下边界，任何违反都会污染后续整个判断链条：

### 约束一：只陈述已发生的变化，不做预测

❌ 错误：`"如果央行继续降息，银行股可能受压"`
✅ 正确：`"央行今日宣布下调存款准备金率0.5个百分点，释放流动性约1万亿元"`

### 约束二：不加入主观判断

❌ 错误：`"这是重大利好，市场将大幅上涨"`
✅ 正确：`"国家统计局公布10月CPI同比上涨0.3%，低于市场预期0.4%"`

### 约束三：不合并来自不同事件的信号

每个 MarketSignal 对应一个独立的事件/变化。不要把多个相关但不同的变化合并成一个信号。

❌ 错误：把"降准"+"LPR下调"+"专项债扩容"合并成一个"货币财政双宽松信号"
✅ 正确：分别提取三个独立的 MarketSignal，让 M3 来判断它们的组合含义

### 约束四：不允许推断受影响的标的超出合理范围

❌ 错误：因为"海运运费下降"就把"所有进出口企业"列为受影响标的
✅ 正确：`affected_instruments` 只列出有直接、明确影响关系的标的

### 约束五：空列表优于低质量信号

如果输入文本中没有有效信号，返回空列表。强行提取出低质量信号比不提取更有害。

## 输出契约

每个 `MarketSignal` 字段的语义承诺：

| 字段 | 承诺 |
|------|------|
| `signal_label` | 20字以内，准确概括信号本质，可以独立阅读 |
| `description` | 完整、客观描述信号，包含所有关键数字 |
| `evidence_text` | 直接引用来源原文，不得改写 |
| `affected_markets` | 只列出有直接影响的市场，不做间接推导 |
| `affected_instruments` | 可以为空，列出时必须有明确因果关系 |
| `signal_direction` | 基于可观察的机制，不是主观预期 |
| `event_time` | 尽量精确到小时，至少精确到日期 |
| `intensity_score` | 见评分指引，1-10 的客观标准 |
| `confidence_score` | 见评分指引，来源可靠性的客观评估 |
| `timeliness_score` | 见评分指引，市场定价程度的客观评估 |
| `logic_frame` | 用"什么变了/如何变/影响谁"三段式描述 |

## 市场差异：A股 vs 港股

### A股信号特征

**政策敏感性极高**
- 政府的表态、政策文件往往是最重要的信号来源
- "定调式"信号（如会议精神、领导人讲话）需要给予较高的 intensity_score
- 政策执行往往有时滞，信号可能早于实际落地数月

**北向资金是重要的资金流向信号**
- 北向资金单日净流入/流出规模超过100亿元应视为较强信号
- 连续多日同向流动更有意义

**情绪主导短期波动**
- A股散户占比高，情绪面信号（来自 MarketSentinel）对短期波动影响大
- M1 本身不处理情绪信号，但要注意标记 `SENTIMENT` 类型信号供后续使用

**ST 制度和退市制度**
- 涉及 ST 风险的信号需要在 warnings 字段标注
- 退市风险标的流动性极差，需特殊处理

### 港股信号特征

**双重定价驱动**
- 港股同时受A股（南向资金）和美股（全球流动性）影响
- 单一市场信号在港股的 `intensity_score` 应适当下调，因为需要两个市场同时共振

**流动性分化严重**
- 蓝筹股（腾讯、阿里等）流动性好，信号有效性高
- 小市值港股流动性极差，即使信号准确也难以执行
- 在 `warnings` 字段标注流动性风险

**AH 溢价信号**
- AH 溢价大幅扩大/收窄本身可以构成信号
- 溢价收窄通常意味着港股相对具有吸引力

**业绩披露季**
- 港股中报（8月）和年报（3月）是重要的事件驱动信号窗口
- 盈利预警（profit warning）需特别关注

## 评分指引

### Intensity Score（强度，1-10）

衡量"变化的幅度和重要性"，参考以下标准：

| 分值 | 含义 | 典型例子 |
|------|------|---------|
| 1-2 | 微弱变化，常规波动范围内 | PMI环比小幅波动±0.1；个股小幅调价 |
| 3-4 | 有意义的变化，超出近期正常区间 | CPI同比读数低于预期0.2个百分点；大盘单日下跌1-2% |
| 5-6 | 显著变化，值得市场关注 | 央行首次降准；主要指数单日涨跌超3%；重要公司业绩大幅低于预期 |
| 7-8 | 重大变化，可能引发结构性影响 | 超预期的央行降息50bp；监管政策重大转向；大型企业违约 |
| 9-10 | 历史级别变化，系统性影响 | 金融危机级别事件；战争/疫情；中美重大政策冲突 |

### Confidence Score（置信度，1-10）

衡量"信号来源的可靠性和确定性"：

| 分值 | 含义 | 典型来源 |
|------|------|---------|
| 1-2 | 传言、未经证实 | 匿名消息、社交媒体爆料 |
| 3-4 | 可信媒体报道，但未官方确认 | 彭博/路透非官方消息，财新调查报道 |
| 5-6 | 较可靠来源，有据可查 | 知名机构研报，经多方交叉验证的信息 |
| 7-8 | 权威来源，信息明确 | 官方媒体（新华社、人民日报）报道，交易所公告 |
| 9-10 | 最高权威，100%确定 | 央行官网公告，证监会正式文件，财政部发文 |

### Timeliness Score（时效性，1-10）

衡量"信号的新鲜程度和市场已反应程度"：

| 分值 | 含义 | 判断依据 |
|------|------|---------|
| 1-2 | 旧信息，市场已充分定价 | 信息发生超过1周，且已有明显的市场反应 |
| 3-4 | 部分已定价，但仍有边际价值 | 1-7天内，市场反应不充分或持续发酵 |
| 5-6 | 近期发生，市场消化中 | 24-72小时内，市场反应尚在进行 |
| 7-8 | 新鲜信息，市场刚开始反应 | 当日发生，市场仍在消化 |
| 9-10 | 刚刚发生，市场几乎未反应 | 过去几小时内发生，交易时间外或刚开盘 |

## 典型案例

### 案例一：好信号 vs 噪音（宏观）

**✅ 好信号（有效）：**
> 原文：「国家统计局数据显示，10月规模以上工业增加值同比增长5.3%，高于市场预期的4.9%，前值4.6%」

```
signal_label: "工业增加值超预期反弹，经济动能边际改善"
signal_type: MACRO
intensity_score: 6
confidence_score: 9
timeliness_score: 8
logic_frame:
  what_changed: "10月工业增加值同比5.3%，高于预期4.9%，连续两月回升"
  change_direction: BULLISH
  affects: ["制造业股票", "周期性行业", "沪深300指数"]
```

**❌ 噪音（无效）：**
> 原文：「某财经博主发文称，年末往往是A股传统旺季，建议积极布局」

→ 这是观点/预测，不是事实变化，不应提取为信号。

---

### 案例二：好信号 vs 噪音（政策）

**✅ 好信号（有效）：**
> 原文：「住建部印发通知，明确商品住宅项目预售资金监管账户内资金可按一定比例用于购买建筑材料和设备，解绑资金比例最高可达40%」

```
signal_label: "住建部松绑房企预售资金监管，40%可提前支取"
signal_type: POLICY
intensity_score: 7
confidence_score: 9
timeliness_score: 8
affected_instruments: ["房地产开发商", "建材行业ETF"]
```

**❌ 噪音（无效）：**
> 原文：「业内人士预计房地产政策还有进一步放松空间」

→ 预期和预测，不是已发生的政策变化。

---

### 案例三：好信号 vs 噪音（资金流）

**✅ 好信号（有效）：**
> 原文：「沪深股通北向资金今日净流入217亿元，为2024年以来单日最大净流入，连续三个交易日净买入超100亿元」

```
signal_label: "北向资金单日净流入217亿创年内新高"
signal_type: CAPITAL_FLOW
intensity_score: 8
confidence_score: 9
timeliness_score: 9
affected_markets: [A_SHARE]
```

**❌ 噪音（无效）：**
> 原文：「外资持续关注中国市场，长期看好中国股市」

→ 这是模糊表态，没有具体数据，无法量化，不构成有效信号。

---

### 案例四：需要拆分的多信号情况

**原文：**「央行今日召开新闻发布会，宣布：（1）下调存款准备金率0.5个百分点；（2）下调LPR利率15bp；（3）下调存量房贷利率约50bp」

→ **应提取3个独立的 MarketSignal，不应合并**

信号1：降准0.5%，释放约1万亿流动性（intensity: 8, type: MACRO）
信号2：LPR下调15bp，历史级别大幅降息（intensity: 8, type: POLICY）
信号3：存量房贷利率下调50bp，直接减轻居民负担（intensity: 7, type: POLICY）

三个信号分别影响不同的市场区域（银行/债市/地产/消费），应由 M3 来判断其组合意义。

---

### 案例五：需要谨慎处理的信号（港股流动性）

**原文：**「某港股上市公司公告控股股东提议私有化，建议注销价每股2.8港元，较停牌前溢价65%」

```
signal_label: "港股某公司私有化提议，溢价65%"
signal_type: EVENT_DRIVEN
intensity_score: 9  # 对该标的极强
confidence_score: 8  # 正式公告
timeliness_score: 9  # 刚刚公告
warnings: ["流动性风险：私有化完成前股价受限；如私有化失败可能大幅回落；需关注控股股东资金能力"]
```

→ 高分不等于可以无脑买入，warnings 是必须的。
